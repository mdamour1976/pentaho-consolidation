It worked!

If you are viewing this the default test perspective successfully loaded.

This perspective defines a javascript function which is associated with a button on the main toolbar.

The main toolbar in PUC is now interactive. You can check the state of any button and toggle enable/disable.

The button has an id of "default.perspective.test.overlay.button". To check the state invoke:
var enabled = window.top.mantle_isToolbarButtonEnabled("default.perspective.test.overlay.button");

To toggle set the state:
window.top.mantle_setToolbarButtonEnabled("default.perspective.test.overlay.button", !enabled);
